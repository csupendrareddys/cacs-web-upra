UI Bars & Naming Conventions

This page lists common "bars" and related UI elements, with their definition, typical placement and examples. The last column suggests the typical HTML element or UI name you may use when building the layout (for accessibility and semantics).

Category Bar Name Definition / Purpose Typical Position Real-World Example HTML / UI Element
Navigation Top Navbar Primary site navigation containing logo and main links. Top of Page Website Header (Home, About, Contact) <nav> (inside <header>)
Navigation Bottom Tab Bar Mobile app navigation for switching main views. Bottom of Screen Instagram or Spotify mobile app <nav role="tablist"> / footer nav
Navigation Sidebar Vertical navigation for complex hierarchies. Left or Right Edge Admin Dashboards, Slack, Gmail <aside> or <nav> (vertical)
Navigation Breadcrumb Bar Shows user location in site hierarchy. Top Left (below nav) Home > Men > Shoes > Running <nav aria-label="breadcrumb">
Navigation Mega Menu Expandable panel showing many options/columns. Drops down from Top Amazon or Best Buy categories <nav> + <div class="mega-menu">
Utility Utility Bar Secondary links (login, language, support). Very Top (above nav) Sign In / language strip <div class="utility-bar"> (or <nav>)
Utility Status Bar OS-level bar showing time, battery, and signal. Extreme Top (Mobile) iPhone/Android status strip OS / Browser UI (not an HTML element)
Utility Toolbar Collection of edit/action tools. Top of Editor MS Word Ribbon, Photoshop Tools <div role="toolbar" class="toolbar">
Utility App Bar Android specific top bar for titles and actions. Top of App Screen Android Action Bar <header role="banner" class="app-bar">
Utility Address Bar Browser input for URLs. Top of Browser Chrome / Safari URL field Browser UI (not an HTML element)
Feedback Snackbar Temporary message with a single action. Bottom Center / Left Message Sent [UNDO] <div class="snackbar" role="status">
Feedback Toast Bar Simple feedback message that disappears automatically. Top Right / Bottom Login Successful popup <div class="toast" role="status">
Feedback Notification Banner Persistent bar for announcements or alerts. Top of Page Free Shipping or Server Maintenance <div class="notification" role="region">
Feedback Progress Bar Visual track indicating task completion status. Middle or Top File download or setup wizard <progress> element or <div role="progressbar">
Input Search Bar Input field for search queries. Header or Center Google Homepage, Header Search <form> + <input type="search">
Input Seek Bar Timeline scrubber for media. Bottom of Video YouTube video timeline <input type="range" class="seek-bar">
Input Slider Bar Range selector for settings. Settings Menu Volume, Brightness, Price Range <input type="range">
Input Scrollbar Navigation tool to view hidden content. Right/Bottom Edge Browser window side Browser-native scrollbar / CSS customization
Legal Footer Bar Catch-all for copyright, sitemap, and policies. Very Bottom Page Copyright © 2024 <footer>
Legal Consent Bar Mandatory notice for cookies/GDPR. Fixed Bottom/Top Accept All Cookies banner <div class="consent-bar" role="dialog">

Tip: When in doubt, use semantic elements (<nav>, <header>, <footer>, <aside>) and ARIA attributes (role, aria-label) to make bars accessible.
